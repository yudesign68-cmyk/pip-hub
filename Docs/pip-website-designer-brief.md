# PiP Website Redesign — Project Brief

## Overview
Convert `partnersinparenting.com.au` from a standalone PiP+ program site into a **hub landing page** that directs users to all available PiP versions. The existing `pip-plus.web.app` remains unchanged for current users.

**Client:** Monash University — Partners in Parenting (PiP) team  
**Deadline:** End of next week (Friday)

## Goals
- Stop new registrations to the outdated PiP+ program
- Create a central, modern hub listing all PiP versions to reduce user confusion
- Keep existing users able to access PiP+ via a clearly signposted link

---

## Pages

### 1. Home (redesigned hub page)
The main deliverable. Must include:

- **Existing account callout box** — prominently placed, separate from main content
  - Heading: *"Do you have an existing account for PiP+?"*
  - Body: directs existing users to `pip-plus.web.app`

- **Hero / Intro section**
  - Heading: *"Welcome to Partners in Parenting (PiP)!"*
  - Subtext: Award-winning, evidence-based online parenting program. Based on decades of research.

- **"What's involved?" section**
  - 3 bullet points: self-reflection questions, interactive modules, evaluation surveys

- **"Where can I access PiP?" section** — core new content
  Cards or grouped list for each PiP version:

  | Version | Audience | Access |
  |---|---|---|
  | PiP (Teens) | Parents of teens ~12–18 | Via headspace (Australia-wide) or Victoria research trial |
  | PiP-Education | Teens with school refusal | Separate program link; new trial late 2026 |
  | PiP Kids | Parents of children ~5–11 | Online, worldwide |
  | PiP Kids-Autism | Parents of autistic children | Contact email (trial ended 2025) |
  | PiP Kids-Education | School refusal, primary age | Trial early 2027, contact email |

- **Evidence-based + Stats section** — treat as one visual chunk (two side-by-side cards)
  - Left card: "Evidence-based parenting program" — research base + link to Resources
  - Right card: "5,000+ families helped" — stats + link to About Us and Contact Us

- **House of Wellness** — place as its own dedicated section
  - "Listen to the stories of previous PiP parents and their teenagers featured in The House of Wellness"

- **"What's in store for you?" section**
  - 5 feature tiles: Personalised feedback / Interactive modules / Fits your lifestyle / Fun & interactive / Free to join

- **Remove:** "Join now" button, any signup flows

## Navigation
- Keep top nav with tabs: Home / Resources / About Us / Contact Us
- **Remove** sign-in button from the hub — users sign in through individual PiP program links
- No new pages required; use a simple template for subpages

---

## Design Direction
- **Style:** Modern, clean — not outdated. No complete rebrand needed.
- **Animations:** Subtle only — fade/dissolve, slide-ins. No heavy interactivity.
- **Density:** Spacious and readable (parents audience — not tech-savvy)
- **Output:** Static HTML files (no Figma required)
- **Tone:** Warm, trustworthy, academic but approachable
